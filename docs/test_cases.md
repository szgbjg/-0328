## Test Case 1: Question Creator
### Input Context
[强直性脊柱炎（AS）是一种常见的慢性炎症性疾病，属于风湿免疫病类。其主要病变部位为骶髂关节、脊柱及外周关节，部分病情严重者可发生脊柱畸形和脊柱强直。AS中的患病率约为0.25%-0.5%，男女患病比例为4:1，女性发病通常较男性缓慢且病情较轻。好发人群为青壮年，其中发病高峰为20-30岁。
AS起病隐匿，早期症状通常是在腰骶部出现钝痛和晨僵，活动后可能减轻。病情进展中，可能出现眼、心脏、肺、肾等多个器官的不同程度的病变。AS不具有传染性。
AS的治疗方法主要为对症治疗，包括药物治疗和必要情况下的手术治疗。同时，保持合理的锻炼和良好的睡姿也是重要的治疗方式之一，能缓解病情并控制病情的发展。AS为一种长期疾病，尚无法治愈，预后因患者的个性差异和早期治疗情况而异。
目前对AS的病因尚不明确，研究表明，发病与遗传、感染、环境、免疫等多个因素相关。具有家族聚集性，某些感染因素以及环境因素可能诱发强直性脊柱炎的发病。尽管AS无法彻底治愈，但通过早期诊断和积极治疗，可以有效的控制重重症状，提高生活质量。]

["问题1", "问题2", "问题3"]

- [ ] 数量为1-5个
- [ ] 无指代性措辞

## Test Case 3: FacetGraph-QA Agent（关键）
### Input
- Q: "阿司匹林的主要禁忌症有哪些？"
- facet: "临床安全"
- refs: 
[
  {"source": "《阿司匹林肠溶片说明书》", "content": "【禁忌】1.对本品或其它水杨酸盐制剂过敏者禁用；2.既往有哮喘史伴有鼻息肉综合征者禁用；3.活动性消化性溃疡患者禁用。"},
  {"source": "《2023年临床心血管用药指南》", "content": "严重肝肾功能不全者、出血体质或伴有活动性出血的患者应避免使用阿司匹林。在妊娠最后三个月应禁用。"}
]

### Expected Output Structure
&lt;think&gt;
&lt;facet=临床安全&gt;
问题拆解:
- S1: 禁忌症定义和范围
- S2: 阿司匹林特定禁忌

证据清单:
- [证据R1:来源=refs:《药品说明书》]
- [证据R2:来源=refs:《临床用药指南》]

推理链:
- P1: 由R1可得...→C1
- P2: 由R2可得...→C2

冲突识别: 无
最终结论摘要: C1, C2, C3
&lt;/think&gt;

[正文回答：无工具痕迹，有出处式表达]

### Validation Rules（关键）
- [ ] think标签格式正确，facet属性正确
- [ ] 无禁止词汇（检索/查阅/工具等）
- [ ] 有明确的证据引用（R1/R2...）
- [ ] 正文有出处式表达（"《XX》提到..."）
- [ ] 证据链完整（证据→推理→结论）
